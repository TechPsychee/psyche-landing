# psyche.solutions

A thought platform exploring the intersections of psychoanalytic theory, technology, and critical inquiry.

**Live:** [psyche.solutions](https://psyche.solutions)

## Structure

```
index.html    ← Main site (single file, everything included)
assets/       ← Images (profile photo, etc.)
CNAME         ← Custom domain config for GitHub Pages
```

## Deploy

Hosted on GitHub Pages with custom domain `psyche.solutions`.

## Adding Essays

1. Create a new HTML file (e.g., `essays/can-a-machine-mentalize.html`)
2. Link to it from the essays section in `index.html`
3. Push to `main` branch — GitHub Pages auto-deploys
