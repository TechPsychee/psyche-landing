The Plot Twistor: how I learned to stop worrying and love the code

For seven years, I practiced as a psychotherapist. Psychodynamic approach, if you're wondering. Yes, that means I occasionally referenced Freud without irony. Yes, I'm aware of how that sounds.

Then one day I thought: what if instead of helping 20 people a week understand their unconscious patterns, I could build tools that reach thousands? What if therapy could scale beyond the fifty-minute hour? What if I'm just rationalizing a mid-career crisis?

"The difference between debugging code and doing therapy? In code, when you find the bug, you can actually fix it. In therapy, sometimes the bug is a feature, and we spend months exploring why the user keeps choosing to run it."

So I went back to school. Got a master's in Information Technologies from Sabancı University. Built AI-powered mental health tools. Started teaching Python at Stanford's Code in Place. Learned that Stack Overflow has basically the same energy as group supervision, just with more passive aggression.

Fun fact: I also have a master's in Forensic Sciences focused on HIV stigmatization and mental health. Because apparently, I enjoy collecting degrees like they're Pokemon cards.
